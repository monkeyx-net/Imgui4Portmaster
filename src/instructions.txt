Press the L1 Button or select the button to start singler player game.

Please note in single player mode. Players are assigned to D-Pad, left stick and right stick.


Controls are left, right and up to jump.

When you start the game use connect to the localhost 127.0.0.1.

Other players can connect to your server using your actual address.

Selecting the R1 Button to change the server IP address before connecting
